# Python Learning

A collection of Python programming exercises and learning materials covering fundamentals through data analysis.

## Topics Covered

### Basics
- Python syntax and basic operations (`ex1.py`, `ex2.py`)
- String manipulation and I/O
- Control flow (conditionals, loops)
- Data structures (lists, dictionaries, tuples)
- Functions and modules (`fun1.py`)
- Exception handling and imports (`imp1.py`)
- File operations (`file.py`, `diary.py`)

### Data Analysis
- CSV data processing (`csv1.py`, `students.csv`)
- pandas fundamentals (`pd1.py`)
- Platform detection and system utilities (`platform.py`)

### Applied Projects
- Blockchain implementation demo (`blockchain.py`)
- Virial theorem calculation (`virial.py`)
